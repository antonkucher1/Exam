Билет №7

Внимательно прочитайте задание и описание предметной области. Ознакомьтесь с последовательностью и условием выполнения частей задания. Выполните четыре задания в соответствии с заданием.
В качестве ответа прикрепите:
Задание 1. Файл с проектом, созданном в одном из CASE-средств Ramus Education или StarUML, а также построенные диаграммы в виде изображений в формате *.png
Задание 2. Исходный проект приложения, созданный с помощью любой среды разработки и исполняемый файл (для запуска проекта без редактора), а также скриншоты процесса работы приложения
Задание 3. Файл в формате word с 3-4 тест-кейсам
Задание 4. Файл в формате word с техническим документом в соответствии с заданием.
Задание 5. Текстовый файл со ссылками:
а) на разделы системы управления проектами с результатами работы
б) на систему контроля версий с кодом приложения.
